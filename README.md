# Healthcare Data Analysis

## Overview
This project focuses on cleaning, validating, and analyzing healthcare data to support accurate reporting and decision-making.

## Tools & Technologies
- SQL (Healthcare KPI analysis)
- Power BI (Interactive dashboards with DAX)
- Tableau (Healthcare analytics dashboard)
- Microsoft Excel (Analytical dashboard & KPIs)

## Dataset
The dataset includes patient visits, diagnoses, lab results, and follow-up information.  
(Uses sample or anonymized data for demonstration.)

## Objectives
- Clean and validate raw healthcare datasets
- Perform SQL-based analysis to generate KPIs
- Build dashboards for trend and performance reporting

## Key Work Performed
- Data cleaning and validation
- SQL joins, aggregations, and filtering
- KPI creation and dashboard design
- Documentation of logic and assumptions

## Outcome
Delivered structured, decision-ready healthcare insights with a strong focus on data accuracy and usability.

## Key Business Questions Answered
- What are the most common diagnosed conditions?
- How do patient visits trend over time?
- What is the follow-up rate across visit types?
- How do lab results vary by test type?
- Which doctors have the highest patient load?
