# Healthcare Data Analysis

## Overview
End-to-end healthcare analytics project focused on data cleaning, KPI generation, and dashboard reporting to support decision-making.

## Tools
SQL · Power BI · Tableau · Excel

## What I Did
- Cleaned and validated healthcare datasets for accuracy and consistency
- Wrote SQL queries to calculate KPIs (follow-up rate, abnormal lab rate, visit trends)
- Built interactive dashboards analyzing trends by diagnosis, visit type, gender, and year

## Key Insights
- Most common diagnosed conditions
- Patient visit trends over time
- Follow-up rates across visit types
- Lab result variations by test type
